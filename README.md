# hueka-legal

Static site hosting the Privacy Policy, Terms of Use, and Support page for the [Hueka](https://apps.apple.com/) iPhone app (color season analysis).

Served via GitHub Pages from the `main` branch root:

- **Index**: https://nugaysamil.github.io/hueka-legal/
- **Privacy Policy**: https://nugaysamil.github.io/hueka-legal/privacy.html
- **Terms of Use**: https://nugaysamil.github.io/hueka-legal/terms.html
- **Support**: https://nugaysamil.github.io/hueka-legal/support.html

## To update

Edit the HTML files, commit to `main`, push. GitHub Pages redeploys automatically within a minute.

When the Privacy Policy or Terms substantively change, bump the **Effective date** at the top of the affected file.

## Reporting issues

Open an [issue](https://github.com/nugaysamil/hueka-legal/issues) — that's the public support channel referenced from the app.
